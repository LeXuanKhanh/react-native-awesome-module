# rn-scan-code-from-image

react native library for scanning barcode, qrcode from image

## Installation

```sh
npm install rn-scan-code-from-image
```

## Usage


```js
import { multiply } from 'rn-scan-code-from-image';

// ...

const result = await multiply(3, 7);
```
